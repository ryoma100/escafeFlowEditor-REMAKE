## Development

### Tauri + Solid + Typescript

This template should help get you started developing with Tauri, Solid and Typescript in Vite.

### Setup

```
pnpm install
```

- for Web Browser

```
pnpm dev
```

- for PC Application

```
pnpm tauri dev
```

## Build

- for Web Browser

```
pnpm build
pnpm serve
```

- for PC Application

```
pnpm tauri build
```
